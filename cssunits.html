Absolute Lenghts

The absolute length units are fixed and a length expressed in any of these will appear as exactly that size.

Mesurement Units Description Example
cm centimeters width:5cm;
mm millimeters width:5mm;
in inches (1in = 96px = 2.54cm) width:5in;
px pixels (1px = 1/96th of 1in) (1in = 96px = 2.54cm) pading:5px;

Relative Units

Relative length units specify a length relative to another length property. Relative length units scales better between different rendering mediums.

Name Description Example
% Percentage Relative to the parent element width:100%
em Relative to the font-size of the element (2em means 2 times the size of the current font) lineheight:5em;
rem Relative to font-size of the root element word-spacing:5rem;
vw Relative to 1% of the width of the viewport font-size: 20vw;
vh Relative to 1% of the height of the viewport font-size: 20vh;
